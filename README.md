## node-mongo

A test project with Node, Express & MongoDB

### Deploy

```shell script
git clone https://github.com/agalustova/node-mongo
cd ./node-mongo
nvm use 14
npm i
```

### Environment

The `.env` file is required, see the [.env.example](.env.example) for details

### Launch (development)

```shell script
npm run dev
```

### Launch (production)

```shell script
npm start
```
